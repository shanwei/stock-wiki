# TSM_台积电（美股）公司核心分析
tags: [美股, 半导体, 晶圆代工, 先进制程, AI芯片, 中风险]

## 核心摘要
台积电（Taiwan Semiconductor Manufacturing Company）是全球最大的晶圆代工厂，CY2025营收28,943亿新台币（约1,125亿美元，+33.8%），净利润11,983亿新台币（+39.6%），先进制程（7nm及以下）占晶圆收入约74%，是AI芯片制造的核心供应商。

## 主营业务拆解

| 业务板块 | CY2025营收占比 |
|----------|---------------|
| 高性能计算（HPC） | ~50% |
| 智能手机 | ~30% |
| 物联网（IoT） | ~8% |
| 汽车 | ~5% |
| 数字消费电子 | ~4% |

| 制程节点 | CY2025营收占比 |
|----------|---------------|
| 3nm | ~20% |
| 5nm | ~35% |
| 7nm | ~19% |
| 先进制程合计（≤7nm） | ~74% |
| 成熟/特殊制程 | ~26% |

- **高性能计算（HPC）**：AI芯片代工是核心增长引擎，为英伟达（GPU）、AMD（GPU）、苹果（M系列芯片）、谷歌（TPU）等客户制造AI芯片，CY2025占营收约50%。
- **智能手机**：为苹果（A系列芯片）、高通（骁龙芯片）等客户代工手机芯片，CY2025占营收约30%。
- **先进制程领先**：3nm已量产，2nm预计2025年量产，技术领先三星和英特尔1-2年。

## 核心竞争力（护城河）
- **先进制程技术领先**：3nm/2nm工艺技术全球领先，良率和性能优于竞争对手（三星、英特尔），是AI芯片制造的唯一可靠供应商。
- **客户粘性极高**：苹果、英伟达、AMD、高通等顶级客户深度绑定，设计生态和制造工艺高度耦合，客户迁移成本极高。
- **规模经济优势**：年产能超过1,600万片12英寸等效晶圆，规模优势带来成本领先和产能保障。
- **研发投入持续高增**：CY2025研发费用2,356亿新台币（+27%），持续投入先进制程和封装技术（CoWoS、3DFabric）。
- **AI需求驱动增长**：AI芯片需求爆发带动HPC业务快速增长，CoWoS先进封装产能成为AI芯片供应的关键瓶颈。

## 管理层与治理
- **董事长刘德音（Mark Liu）**：2024年退休，由魏哲家（C.C. Wei）接任董事长兼CEO。
- **CEO魏哲家（C.C. Wei）**：技术出身，主导先进制程战略，带领公司抓住AI机遇。
- **治理结构**：台湾证券交易所和NYSE双重上市，外资持股比例较高，治理透明度较好。
- **地缘政治风险**：总部位于台湾，面临地缘政治不确定性，已在美国（亚利桑那州）、日本（熊本）、德国布局海外工厂分散风险。

## 行业地位与竞品对比
- **市场份额**：全球晶圆代工市场占有率超过60%，先进制程（7nm及以下）市场占有率超过90%。
- **竞品对比**：
  - 与[[NVDA_英伟达_美股/wiki/index.md|英伟达]]：英伟达是台积电最大客户之一，台积电为英伟达代工数据中心GPU。
  - 与[[GOOG_谷歌_美股/wiki/index.md|谷歌]]：谷歌自研TPU芯片由台积电代工。
  - 三星晶圆代工（市占率约15%）和英特尔代工服务（IFS）是主要竞争对手，但在先进制程技术上落后台积电1-2年。
- 行业分析详见：[[02_行业_AI/wiki/index.md]]

## 核心逻辑总结
1. **AI芯片制造的垄断者**：全球先进制程晶圆代工市场台积电占据超过90%份额，AI芯片（GPU、TPU、ASIC）几乎全部由台积电制造，是AI基础设施的核心供应商。
2. **先进制程技术护城河深厚**：3nm/2nm工艺技术领先竞争对手1-2年，客户粘性极高，竞争对手短期内难以追赶。
